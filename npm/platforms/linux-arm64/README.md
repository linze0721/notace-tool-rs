# @not-ace-tool-rs/linux-arm64

Platform-specific binary for not-ace-tool-rs (Linux ARM64).

## Overview

This package contains the pre-built binary for Linux ARM64. It is automatically installed as an optional dependency when you run `npx not-ace-tool-rs` on Linux ARM64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx not-ace-tool-rs --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- Linux arm64/aarch64
